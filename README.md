1、jquery的概念
问题：
使用js完成对网页的操作，代码的书写量比较大。js中对网页操作提供的方式过
于简单，造成获取和操作HTML元素对象过于麻烦。
解决:
使用jQuery
结论：
其实jQuery就是讲常用的js操作封装了起来。jQuery是js的一个框架。
内容:
jQuery本质上还是js，基本语法和js一致。将js的DOM操作重新封装
整合了。
学习:
如何使用JQuery完成网页操作。
